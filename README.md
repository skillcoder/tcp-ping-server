# TCP Ping Server

### GET
```
mkdir -p ~/go/src/github.com/skillcoder
cd ~/go/src/github.com/skillcoder
git clone https://github.com/skillcoder/tcp-ping-server.git
cd tcp-ping-server
```

### BUILD
```
go build
```

### RUN
```
TCPING_SERVICE_LISTEN=127.0.0.1:8888 ./tcp-ping-server
```
